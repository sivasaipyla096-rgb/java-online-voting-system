\# Java Online Voting System



A secure, role-based online voting system developed using Java, JSP, Servlets, and MySQL.  

This project demonstrates backend development, authentication, and database integration.



\## Features

\- Admin login and candidate management

\- Voter registration and authentication

\- Secure vote casting (one voter, one vote)

\- Real-time vote counting

\- Result declaration dashboard



\## Tech Stack

\- Java

\- JSP \& Servlets

\- Apache Tomcat

\- MySQL

\- HTML, CSS



\## How to Run

1\. Clone the repository

2\. Import project into IntelliJ IDEA / Eclipse

3\. Configure Apache Tomcat server

4\. Import MySQL database (schema provided if available)

5\. Run the project on Tomcat
